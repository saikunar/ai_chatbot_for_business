# 🤖 AI Business Chatbot

This project is an AI-powered chatbot designed for businesses to automatically respond to customer queries.

---

## 🚀 Features

- Train chatbot with business information
- Answer customer questions instantly
- Simple UI using Streamlit
- Fast backend using FastAPI
- Runs locally using Ollama (no paid API required)

---

## 🧠 How It Works

1. Enter business details (e.g., restaurant info)
2. Train the chatbot
3. Ask questions
4. AI responds based on your data

---

## 🛠️ Tech Stack

- Python
- FastAPI
- Streamlit
- Ollama (LLM)
